## Installation

1. Clone the project

```bash
git clone [your-github-url]
cd laravel-flowbite
```

2. Install dependencies

```bash
composer install
npm install
```

3. Configure environment

```bash
cp .env.example .env
php artisan key:generate
```

4. Configure database in .env

5. Migrate database

```bash
php artisan migrate --seed
```

6. Compile assets

```bash
npm run dev
```

7. Start server

```bash
php artisan serve
```
